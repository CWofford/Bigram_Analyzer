To run this app, seperately run the front end with npm run dev, 
and the back with python3 main.py only after activating a virtual enviroment 
source venv/bin/activate. 

To use the app open the frontend which I have been running locally at http://127.0.0.1:5173/
then input a text input of at least 3 character 2 of which must be non-space chars
then click the analyze button to see the statistics of the number of words,
bigrams, and unique bigrams. Below that is the Bigram histogram, followed by the 
individual stats of the occurance rate of each bigram
